# Night Killers

Simple static-first Next.js website for the RuneScape clan Night Killers.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Motion

## Local development

```powershell
npm.cmd install
npm.cmd run dev
```

Then open `http://localhost:3000`.

## Content

Replace placeholder clan copy, Discord URL, apply URL, and contact email in:

- `src/content/site.ts`
- `src/content/members.ts`
- `src/content/highlights.ts`

## Implementation docs

The implementation pack lives in `docs/implementation/night-killers/`.
