# Page Specs

## Home

### Hero

- Left side: eyebrow, large clan title, tagline, short description
- CTA row: Apply and Members
- Supporting chips for playstyle highlights
- Right side: logo/crest presentation card with supporting micro-copy

### Clan Intro

- One larger lead paragraph
- Two supporting paragraphs
- Tone: confident, community-first, not overly corporate

### Stats

- Three key figures
- Each stat gets a short clarifying note
- Cards stay equal height on desktop and stacked on small screens

### Feature Grid

- Four cards
- Focus on PvP, events, leadership, and organized communication

### Members Preview

- Show leadership first
- Include role, short bio, and specialties if present
- Link to the dedicated Members page

### Final CTA

- Large closing panel
- One sentence about joining
- Apply and Discord actions

## Members

- Large page hero
- Leadership section first with stronger card emphasis
- Core roster section below
- All cards follow the same visual language as Home

## Apply

- Explain the kind of player the clan wants
- Requirements displayed as compact cards
- Process shown in three steps
- Finish with a strong external CTA
- No inline form submission in v1

## Contact

- Minimal page hero
- One card for Discord
- One card for email
- One card for navigation back into the site
- Keep this route lightweight and direct
