# Style Guide

## Color Direction

- Background base: `#03060d` to `#07111f`
- Surface base: semi-transparent navy panels
- Primary accent: bright blue around `#2f85ff`
- Secondary accent: pale ice-blue around `#8fd4ff`
- Ember accent: orange-red around `#ff6a2f`
- Text primary: near-white
- Text muted: slate-blue gray

## Typography

- Headings: condensed, bold, uppercase leaning
- Body: clean sans serif with high readability
- Use letter spacing on small labels and CTA text
- Keep long paragraphs narrow for readability

## Layout

- Mobile-first
- Max content width around `1280px`
- Consistent horizontal padding per breakpoint
- Use stacked sections on phones
- Open to two-column and split-layout sections on desktop

## Surfaces

- Rounded panels with soft borders
- Mild blur for important surfaces only
- Use layered gradients rather than flat fills
- Reserve the strongest glow for CTAs and hero/logo surfaces

## Buttons

- Primary: blue gradient, bright text, soft outer glow
- Secondary: dark surface, subtle border, hover brightness
- Ghost: transparent or lightly tinted for secondary actions

## Cards

- Slight lift on hover
- Top-left icon or badge
- Tight headline, short paragraph, optional supporting note
- Avoid overloading cards with secondary actions

## Motion

- Hero content: short fade/slide in
- Cards: small reveal on scroll
- Hover: minimal translate and glow change
- Mobile menu: fade overlay and sliding panel
- Keep all animations short and avoid continuous movement

## Mobile Rules

- Navigation collapses into off-canvas menu
- CTA buttons stay full-width or comfortably tappable
- Split hero becomes single column
- Text stays readable without pinch-zoom
